# README
## Aqui vai uma breve explicação de cada página criada.
*Este site `index.html` é uma breve apresentação de um site pessoal. Nele, a autora dá boas-vindas aos visitantes, informando que o site contém informações sobre mim e minhas receitas favoritas, e destaca que há links para as seções “Sobre mim” e “Receitas”.*

*Já o site `sobremim.html` descreve a autora do site se apresentando citando o seu nome e sobrenome, sua idade, onde estuda, seu endereço e seus estilos preferidos de música.*

*O site `receitas.html` a autora menciona o passo a passo de duas receitas de suas comidas prediletas com o modo de preparo*
